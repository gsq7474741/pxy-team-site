# ICP 备案

在中国大陆提供互联网服务必须完成 ICP 备案。

## 备案要求

::: danger 法规要求
根据《互联网信息服务管理办法》，未经备案不得从事互联网信息服务。
未备案网站将无法通过域名访问阿里云服务器。
:::

### 需要备案的情况

- 网站服务器在中国大陆
- 域名解析指向大陆服务器

### 不需要备案的情况

- 服务器在香港或海外
- 仅供境外用户访问

## 备案类型

| 类型 | 说明 |
|------|------|
| 首次备案 | 主体和域名均未备案 |
| 新增网站 | 主体已备案，新增域名 |
| 新增接入 | 网站已在其他服务商备案 |
| 变更备案 | 修改已备案信息 |
| 注销备案 | 注销已有备案 |

## 准备材料

### 个人备案

| 材料 | 要求 |
|------|------|
| 身份证 | 正反面照片 |
| 手机号 | 本人实名认证 |
| 邮箱 | 用于接收通知 |
| 域名证书 | 在域名控制台下载 |
| 核验照片 | 按要求拍摄 |

### 企业备案

| 材料 | 要求 |
|------|------|
| 营业执照 | 彩色扫描件 |
| 法人身份证 | 正反面照片 |
| 网站负责人身份证 | 正反面照片 |
| 授权书 | 如负责人非法人 |
| 域名证书 | 实名认证后下载 |

## 备案流程

### 1. 验证备案类型

1. 登录 [阿里云备案系统](https://beian.aliyun.com)
2. 输入主体信息
3. 系统判断备案类型

### 2. 填写主体信息

**个人主体**
- 姓名、证件号
- 通讯地址
- 联系方式

**企业主体**
- 企业名称
- 证件类型和号码
- 法人信息
- 通讯地址

### 3. 填写网站信息

| 字段 | 示例 |
|------|------|
| 网站名称 | 彭教授研究团队官网 |
| 域名 | prof-peng.team |
| 网站类型 | 非经营性 |
| 网站内容 | 学术研究展示 |
| 网站语言 | 中文/英文 |
| 前置审批 | 无 |

::: warning 网站名称要求
- 不能包含"中国"、"中华"等词
- 个人网站不能使用公司性质名称
- 需与网站实际内容相符
:::

### 4. 上传材料

按提示上传：
- 证件照片
- 核验照片（阿里云 APP 拍摄）
- 域名证书

### 5. 初审

阿里云初审：
- 1-2 个工作日
- 可能电话核实
- 修改问题后重新提交

### 6. 短信核验

部分省份需要：
- 工信部发送验证短信
- 24 小时内完成验证
- 验证网址：[beian.miit.gov.cn](https://beian.miit.gov.cn)

### 7. 管局审核

- 提交至省通信管理局
- 审核时间：3-20 个工作日
- 各省不同，浙江约 5 天

### 8. 备案完成

- 收到备案号
- 邮件/短信通知
- 可在工信部网站查询

## 公安备案（可选）

部分地区要求公安备案：

1. 登录 [全国公安机关互联网站安全管理服务平台](http://www.beian.gov.cn)
2. 注册并登录
3. 新办网站申请
4. 填写网站信息
5. 等待审核

## 备案注意事项

### 时间规划

| 阶段 | 时间 |
|------|------|
| 材料准备 | 1-2 天 |
| 阿里云初审 | 1-2 个工作日 |
| 短信核验 | 1 天 |
| 管局审核 | 3-20 个工作日 |
| **总计** | **约 1-4 周** |

### 常见驳回原因

1. **证件问题**
   - 照片模糊
   - 证件过期
   - 信息不一致

2. **网站信息问题**
   - 名称不合规
   - 内容与类型不符
   - 域名未实名

3. **负责人问题**
   - 电话无法接通
   - 信息与证件不符

### 备案期间

- 网站可暂时关闭
- 或使用临时页面（显示备案中）

```html
<!DOCTYPE html>
<html>
<head>
  <title>网站备案中</title>
</head>
<body>
  <h1>网站正在备案中，敬请期待</h1>
  <p>预计 1-2 周内完成</p>
</body>
</html>
```

## 备案变更

信息变更需及时更新：

- 主体信息变更（地址、电话）
- 网站信息变更（负责人）
- 域名变更

## 备案注销

不再使用的备案需注销：

1. 登录备案系统
2. 选择"注销备案"
3. 填写注销原因
4. 提交审核
