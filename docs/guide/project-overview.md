# 项目概述

PXY Team Site 是一个学术团队展示网站，采用现代化的全栈架构设计。

## 项目背景

该网站用于展示彭教授研究团队的：

- **研究方向** - 团队主要研究领域
- **团队成员** - 导师、博士生、硕士生、校友
- **学术成果** - 论文、专利、竞赛获奖
- **新闻动态** - 团队新闻和公告
- **招聘信息** - 开放岗位和加入指南

## 技术选型理由

### 前端：Next.js 15

- **SSR/SSG 支持** - 良好的 SEO 表现
- **App Router** - 现代化的路由系统
- **React 19** - 最新的 React 特性
- **ISR** - 增量静态再生，平衡性能和实时性

### 后端：Strapi 5

- **Headless CMS** - 内容与展示分离
- **可视化管理** - 非技术人员也能管理内容
- **多语言支持** - 原生 i18n 能力
- **API 自动生成** - REST/GraphQL API 开箱即用

### 构建工具：pnpm + Turbo

- **Monorepo** - 前后端统一管理
- **高效缓存** - 增量构建，速度快
- **依赖共享** - pnpm 的硬链接机制节省磁盘空间

### 部署：阿里云函数计算

- **Serverless** - 按需计费，成本低
- **自动扩缩** - 应对流量波动
- **国内访问快** - 适合目标用户群体

                                              │
                              ┌───────────────┴───────────────┐

                    ┌─────────────────┐            ┌─────────────────┐
                    │   PostgreSQL    │            │   阿里云 OSS    │
                    │   (云数据库)     │            │   (媒体存储)    │

## 数据流

1. **用户访问前端** → Next.js 服务端渲染
2. **前端请求数据** → Strapi REST API
3. **Strapi 查询** → PostgreSQL 数据库
4. **媒体文件** → 阿里云 OSS CDN 分发

## 主要功能模块

| 模块 | 路由 | 说明 |
|------|------|------|
| 首页 | `/` | 研究方向概览 + 最新新闻 |
| 研究方向 | `/research` | 研究方向列表和详情 |
| 团队成员 | `/members` | 成员列表和个人页面 |
| 学术成果 | `/publications` | 论文、专利、获奖 |
| 新闻动态 | `/news` | 新闻列表和详情 |
| 加入我们 | `/join` | 招聘岗位和详情 |
| 联系我们 | `/contact` | 联系方式 |

## 国际化

网站支持中文和英文两种语言：

- **Cookie 持久化** - 用户语言偏好保存 30 天
- **前端翻译** - 基于 next-intl
- **后端翻译** - Strapi 原生 i18n
